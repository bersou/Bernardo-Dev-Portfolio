Um portfólio de elite com estética **Industrial-Brutalista** e tema **Fire Blast**, desenvolvido para destacar competências em desenvolvimento web e qualidade industrial.

## 🔥 Diferenciais Visuais
- **Noon Wave Animation:** Fundo dinâmico com ondas de calor pulsantes.
- **Cinematic Fire Background:** Vídeo imersivo de chamas em loop infinito.
- **Estética BPCI:** Design focado em autoridade técnica e segurança industrial.
- **Glassmorphism:** Cards interativos com efeito de vidro e brilho neo-industrial.

## 🛠️ Tecnologias Utilizadas
- **Next.js 16 (App Router)**
- **React 19**
- **Tailwind CSS v4**
- **Framer Motion** (Animações dramáticas e micro-interações)
- **Lucide React** (Iconografia técnica)

## 🚀 Como Executar o Projeto

### Pré-requisitos
Certifique-se de ter o **Node.js** instalado em sua máquina.

### 1. Instalação
Clone o repositório e instale as dependências:
```bash
npm install
```

### 2. Rodar em Desenvolvimento
Para iniciar o servidor local:
```bash
npm run dev
```
O projeto estará disponível em `http://localhost:3000`.

### 3. Build de Produção
Para gerar a versão otimizada para deploy:
```bash
npm run build
```
